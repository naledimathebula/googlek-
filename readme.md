# 📝 Google Keep Clone

A simplified Google Keep Clone built using HTML5, CSS3, and JavaScript. This application allows users to create, display, archive, and manage notes through an intuitive and responsive interface inspired by Google Keep.

---

## 📖 Project Overview

This project was developed to strengthen core web development skills by recreating essential Google Keep functionality using vanilla JavaScript.

Users can:

* Create notes
* View notes
* Archive notes
* Organize information efficiently
* Interact with a clean and responsive user interface

The project demonstrates proficiency in HTML structure, CSS styling, JavaScript functionality, and user experience design.
